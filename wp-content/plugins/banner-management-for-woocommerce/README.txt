== Frequently asked questions ==

= In which WordPress version this Plugin is compatible? =

It is compatible from 4.0 to 5.5.1 WordPress version.

= In which WooCommerce version this Plugin is compatible? =

It is compatible for 4.0 and greater than WooCommerce plugin

= Is your plugin compatible with Astra theme? =

Yes, our plugin is compatible with Astra theme and you can add below CSS code which is provided by the <strong>Astra support team</strong> to show the banner in mobile at top place.
Add this css on your themes style or any css: <strong>.wbm_banner_image { order: 1;}</strong>
